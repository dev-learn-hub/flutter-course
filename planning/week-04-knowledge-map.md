# 第4周知识点体系大纲

> 本周主题与目标（一句话）：掌握 Dart 变量、类型、操作符、字符串、函数与闭包，能编写简单 Dart 程序（含学生成绩管理 CLI）。

---

## 一、变量与类型系统

| 知识点 | 简要说明 | 学生应能… |
|--------|----------|-----------|
| var / dynamic / final / const | 四种声明方式与适用场景 | 正确选用并解释区别 |
| 类型注解与类型推导 | 显式类型 vs 隐式推导 | 写出带类型注解与纯推导的示例 |
| 基本类型 int/double/bool | 字面量与运算 | 做算术、比较、逻辑运算 |

---

## 二、数据类型与集合（List / Map / Set）

| 知识点 | 简要说明 | 学生应能… |
|--------|----------|-----------|
| List 创建与操作 | 字面量、length、add/remove、遍历 | 创建列表并完成增删查、迭代 |
| Map 创建与访问 | 字面量、[]、keys/values、putIfAbsent | 用 Map 存键值对并查询 |
| Set 与唯一性 | 去重、union/intersection/difference | 用 Set 做简单集合运算 |

---

## 三、操作符与字符串

| 知识点 | 简要说明 | 学生应能… |
|--------|----------|-----------|
| 算术 / 关系 / 逻辑 / 赋值 | + - * / ~/ %、==、&& \|\| !、??= | 在程序中正确使用 |
| 条件与空值操作符 | 三元、??、.. 级联 | 写条件表达式与链式调用 |
| 字符串创建与插值 | 单/双/三引号、$var、${expr} | 用插值拼接字符串 |
| 字符串常用方法 | length、split、trim、replaceAll 等 | 对字符串做常用处理 |

---

## 四、函数与参数

| 知识点 | 简要说明 | 学生应能… |
|--------|----------|-----------|
| 函数定义与调用 | 有返回值/void、=> 箭头函数 | 定义并调用函数 |
| 必需/可选/命名/默认参数 | (a,[b]), {name, age}, default | 写出多种参数形式的函数 |
| 函数为一等公民 | 函数作变量、作参数、作返回值 | 写出高阶函数示例 |

---

## 五、闭包与作用域

| 知识点 | 简要说明 | 学生应能… |
|--------|----------|-----------|
| 闭包概念 | 内层函数捕获外层变量 | 解释并写一个闭包示例 |
