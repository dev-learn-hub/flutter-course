## 课程目标

本课程教授基于 Dart 语言和 Flutter 框架的移动应用开发，使学生能够：

1. 搭建完整的 Flutter 多平台开发环境  
2. 掌握 Dart 编程语言的语法和特性  
3. 掌握 Flutter 框架的核心组件和布局系统  
4. 独立开发能在 Android / iOS 平台部署的移动应用  
5. 具备团队协作开发大型应用的能力  
6. 了解开源文化与软件行业发展趋势

| 第 7 周 | [Dart Web 与 Flutter Web](week-07/) | DOM 操作、Flutter Web、响应式设计 |

| 第 12 周 | [Flutter 动画与交互](week-12/) | Animation、隐式/显式动画、Hero 动画、页面切换 |
| 第 13 周 | [应用架构与状态管理](week-13/) | Provider、GetX、Bloc、架构设计 |
| 第 14 周 | [项目实践（一）](week-14/) | Git、GitHub、代码规范、CI/CD、项目管理 |
| 第 15 周 | [项目实践（二）](week-15/) | 小组协作、产品设计、原型制作、需求分析 |
| 第 16 周 | [项目实践（三）](week-16/) | 项目展示、互评、应用打包、发布流程 |

## 课程特色

- **全实践导向**：54 学时在机房实操，边学边练  
- **内容新颖**：采用当前 Flutter 技术栈，紧跟行业  
- **思政融合**：融入国产软件、开源贡献等价值观  
- **注重应用**：最后三周为项目实践，可产出可上线产品  
- **团队协作**：通过小组项目锻炼沟通与协作能力  
- **对接市场**：技术栈对应移动开发市场主流需求
