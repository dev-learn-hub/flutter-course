# 学生成绩管理系统（命令行版）

第4周案例：演示 Dart 变量、Map、List、函数与字符串插值。

## 运行

在项目根目录执行：

```bash
dart run bin/main.dart
```

或（Dart 3+）：

```bash
dart run .
```

## 说明

- 使用 `Map<String, List<int>>` 存储学生姓名与成绩列表。
- 函数：`addScore`、`printAll`、`printStudent`、`averageAll`、`highestAll`、`lowestAll`、`printSortedByAverage`。
- 对应大纲：变量与类型、List/Map 操作、函数定义与调用。
