# 第15周 案例：产品需求与协作示例

第15周侧重**产品设计、需求分析、原型与小组协作**，本目录提供 PRD/用户故事示例文档，供课堂与作业参考。无 Flutter 可运行代码；可与本课程仓库、GitHub Projects、Figma 等配合使用。

## 内容说明

| 文件 | 说明 |
|------|------|
| **prd_example.md** | 简易产品需求文档示例（含用户故事、功能列表、验收思路） |
| **README.md** | 本说明 |

## 使用建议

1. 结合 [第15周大纲](https://github.com/dev-learn-hub/flutter-course/tree/main/week-15.md) 中的 PRD 模板与用户故事卡格式，完善或仿写自己的 PRD。
2. 用 Figma 等工具做低保真/中保真原型，对应 PRD 中的主要页面与流程。
3. 在小组项目仓库中启用 GitHub Projects，将任务分解后录入看板，与 Sprint 计划配合。
4. 参考大纲中的 Scrum 流程（Sprint Planning、Daily Standup、Review、Retrospective）进行协作演练。

## 与课程大纲的对应

- 产品设计与需求分析（PRD、用户故事）
- 产品原型设计（低保真/中保真/高保真、Figma）
- 小组协作与分工（角色、Scrum、任务分解、GitHub Projects）
